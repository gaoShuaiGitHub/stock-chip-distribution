# 股票筹码分布查询

查询指定股票的筹码累计分布，输出交互式 HTML 折线图（含 OBV + 量价异动标注）。

## 运行命令

```bash
cd ~/.openclaw/workspace/skills/stock-chip-distribution/scripts
# 默认代理端口 7897，无需手动指定
python3 chip_chart.py --symbol OKLO --days 365
python3 chip_chart.py --symbol 00700.HK --days 180
python3 chip_chart.py --symbol 600519.SS --days 365
# 如需指定其他代理
python3 chip_chart.py --symbol OKLO --days 365 --proxy socks5://127.0.0.1:7897
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--symbol, -s` | 股票代码（必需） | - |
| `--days, -d` | 查询天数 | 365 |
| `--start-date` | 起始日期 YYYY-MM-DD | 自动计算 |
| `--output, -o` | 输出 HTML 路径 | `./{symbol}_筹码分布.html` |
| `--proxy` | SOCKS5 代理地址 | `socks5://127.0.0.1:7897` |

## 股票代码格式

- **美股**: `OKLO`, `AAPL` → 自动加 `.US`
- **港股**: `00700.HK`, `09988.HK`
- **A 股**: `600519.SS`（上证）, `000858.SZ`（深证）

## 输出内容

**主图**：
- 布局：左侧图表（筹码+股价），右侧 tooltip 详情面板，**高度一致**
- X 轴：日期（每日数据）
- 左轴（0–100%）：10 条筹码累计占比折线（赤橙黄绿青蓝紫等色）
- 右轴（价格$）：收盘价走势折线
- **交互**：点击图例单独高亮某条线，双击恢复全部

**OBV 副图**：
- OBV 能量潮指标，跟涨绿色/跟跌红色
- OBV 持续上升 = 资金持续流入

**量价异动日标注**：
- 触发条件：放量 > 2.5× 20日均量，或涨跌 > 8%
- 出现在 OBV tooltip 里（⚠️ 标记）
- 页面底部附异动日汇总表

**实战用法**：
- 放量暴涨日 = 机构可能大量买入的节点
- 放量暴跌日 = 机构可能出货的节点
- 看当前股价是否在「放量暴涨日收盘价」之上——若长期在之下无法收复，说明机构可能已撤退锁利

## 数据源与算法

- **数据源**: Yahoo Finance API
- **代理**: SOCKS5（默认 `socks5://127.0.0.1:7897`，可通过 `--proxy` 覆盖）
- **筹码算法**: 每日成交量按价格区间均分，累加归一化（10 档，从历史最低到最高）
- **OBV**: 经典能量潮（收涨加量，收跌减量）
- **异动检测**: 20日均量对比 + 8% 涨跌幅阈值

## 故障排除

**连接超时**: 检查代理是否可用

**无数据返回**: 确认股票代码正确且公司已上市

**Proxy 错误**: 确保 `--proxy` 格式为 `socks5://host:port`
